# User Manual
## Digital Transformation System
### Baikal Fish Processing Plant

## System Overview

The AI-Baikal system provides real-time monitoring and analytics for fish processing operations, including production efficiency, quality control, and inventory management.

## Getting Started

### Login Procedure

1. **Access the System**
   - Open web browser and navigate to: `https://dashboard.baikal-fish.ru`
   - Enter your username and password
   - Complete two-factor authentication if required

2. **Dashboard Navigation**
   - Main Dashboard: Overview of all key metrics
   - Production Module: Real-time production monitoring
   - Quality Control: Quality metrics and alerts
   - Inventory Management: Stock levels and tracking

## Production Module
